Taylor Series Visualizer

This code visualizes the transition of a polynomial into the exponential function as infintely many terms are added. In this project, an interactive slider would be used to control the addition of terms and the results would be visualized on a graph.

Define a set amount of terms which would serve a limit to the slider's range.
Generate a set of coefficients for the terms (Taylor coefficients used here)
Generate the linear combination of the terms 
Visualize it on a graph

Quantum State Visulaizer

A quantum state is defined by the superposition of the possible basis vector states multiplied by complex amplitudes. This state contains all the information about the physical system it describes and these information can be extracted as probabilities. This project achieves the following aims:

Represent a quantum state in the form of its basis vector
Normalize the state vector to achieve the probability distribution condition.
Determine the projection matrix of the state and understand it significance
Rotate the basis and change it to see its effect on the system
Visualize it on a Bloch Sphere
Measure the state
